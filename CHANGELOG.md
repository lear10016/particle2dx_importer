# Changelog

## 1.0.0 - 2026-06-25

- Initial Asset Store candidate.
- Added editor menu conversion for Cocos2d-x ParticleDesigner plist files.
- Added one-click folder batch conversion for `.plist` files.
- Added headless CLI conversion for batch workflows.
- Added support for embedded base64 texture data and external PNG, JPEG, and WebP textures.
- Added gravity mode and radius mode mapping to Godot `GPUParticles2D`.
